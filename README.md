## Getting started

First, install all the dependencies:

```bash
npm i
# or if you're using Bun
bun i
```

To run the dev server:

```bash
npm run dev
# or if you're using Bun
bun dev
```

To export and preview the production build:

```bash
npm run build
npm run preview

# or if you're using Bun
bun run build
bun preview
```
